# 🚀 SODEX OS V5 - The Architect Edition

**Sodex OS V5** is a powerful, 10-language supported desktop assistant built with Python. Designed with a focus on high performance, modern UI, and AI integration. Developed by an 11-year-old coder (Sodex). 🏎️💨

---

## 🔥 Key Features

* 🌍 **Multi-Language Support:** Seamlessly switch between 10 different languages (English, Turkish, Spanish, and more!).
* 🤖 **AI Background Removal:** Integrated `rembg` support for instant image processing.
* 🎨 **Modern UI:** Built with `customtkinter` for a sleek, dark-mode compatible interface.
* ⚡ **System Tools:** Built-in network speed tests and hardware optimization tools.
* 🛠️ **Professional Suite:** Advanced file management and user-friendly shortcuts.

---

## 🛠️ Installation

To run Sodex OS on your local machine:

1.  Ensure you have Python 3.13+ installed.
2.  Install the required dependencies:
    ```bash
    pip install customtkinter rembg pillow psutil speedtest-cli
    ```
3.  Launch the application:
    ```bash
    python sodexos.py
    ```
